# TLS_DSS/model_submodule

The model submodule of the TLS DSS is where the predictive neural network is 
stored. This NN is engaged by the knowledge engine, but is built on data in the
data submodule.

## model_submodule/model_dev/

This folder contains model development for the arNN, including data that were
produced from the NASA-NW repository workflow (`y_SRM_data_export` {targets} 
group). Since the data are minimal, we're storing them here in this repository.
